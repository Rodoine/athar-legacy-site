# Athar Legacy — Intergenerational Legacy Technology

Official website for Athar Legacy. Built with React + Vite.

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import this repo
3. Click Deploy (no config needed, Vercel detects Vite automatically)
4. Add custom domain: atharlegacy.com

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
